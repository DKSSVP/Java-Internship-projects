Description:
-------------
This project is a simple word-guessing game.
The computer randomly selects a word, and the player must guess it one letter at a time.

How to Run:
------------
1. Compile the code:
   javac HangmanGame.java

2. Run the application:
   java HangmanGame

Features:
----------
- Random word selection from a predefined list
- Displays word with hidden letters (underscores)
- Tracks wrong guesses (max 6 attempts)
- Shows win/loss message at the end

Words Used:
------------
- java
- program
- keyboard
- internship
- object

Note:
------
- Case-sensitive: enter letters in lowercase.
- Can be extended to use file input or user-defined words.
